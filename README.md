Ola Ride Analytics Dashboard

Project Overview

This project analyzes a synthetic Ola ride-booking dataset containing more than 20,000 records. The dataset was generated with the assistance of ChatGPT for educational and portfolio-building purposes and simulates real-world ride-hailing operations.

The project demonstrates the complete data analytics workflow:

* Data preparation and cleaning
* SQL-based data analysis using MySQL
* KPI generation and business insights
* Interactive dashboard development in Power BI

Dashboard Sections

* Overview
* Vehicle Type Analysis
* Revenue Analysis
* Cancellation Analysis
* Rating Analysis

Key Insights

* Booking trends and ride completion rates
* Revenue performance across vehicle categories
* Customer and driver cancellation patterns
* Customer and driver rating analysis
* Payment method preferences

Tools Used

* MySQL
* Power BI
* Microsoft Excel
* ChatGPT (for synthetic dataset generation)

Dataset

The dataset used in this project is synthetic and intended solely for learning and demonstration purposes. It does not contain any real customer or business data.
